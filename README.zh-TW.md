# 泠靈小工具

**專為樂團譜務設計的模板化分譜管理系統。**

[English](README.md) | [繁體中文](README.zh-TW.md)

## 專案簡介

**泠靈小工具** 是一套桌面應用程式，旨在解決樂團行政與譜務人員 (Music Librarians) 在處理「分譜檔案分割」後的命名痛點。

從製譜軟體匯出的分譜檔名往往包含冗餘資訊且格式混亂。本工具不僅僅是批次改名，而是提供了一套靈活的**「雙層級模板系統」**，讓您能針對不同曲目設定專屬的命名邏輯，快速將雜亂的檔案轉換為標準化的歸檔格式。

## 核心功能

### 1. 自訂檔名模板 (Custom Templates)
使用者可透過變數語法（如 `{樂器}`, `{編號}`, `{曲名}`）自由定義檔名格式，無需受限於預設規則。

### 2. 雙層級獨立架構 (Dual-Layer System)
本工具設計了兩種模板層級，以應對包含多首曲目的複雜音樂會製作：

* **通用大模板 (Master Template):**
    預設套用於所有檔案的基礎規則。
    
* **曲目小模板 (Track-Specific Override):**
    針對特定曲目或檔案群組設定的專屬規則。
    * **獨立覆寫機制:** 當您為某些檔案指定了「小模板」時，該模板將**完全取代**「大模板」，而非僅是附加在後。這讓您能在同一次作業中，同時處理格式截然不同的多首曲目。

## 下載與使用教學

1.  前往 **[Releases](../../releases)** 頁面下載最新版 `LingLingSuite.exe`。
2.  開啟程式，匯入包含分譜檔案的資料夾。
3.  **設定大模板:** 在上方欄位輸入預設規則（例如：`{編號}. {樂器}`）。
4.  **設定小模板 (選用):** 若資料夾內包含不同曲目，選取特定檔案並右鍵設定獨立模板（例如：`Beethoven - {樂器}`）。
5.  確認預覽列表無誤後，點擊「執行處理」。
